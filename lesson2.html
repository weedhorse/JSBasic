1. Дан код: 
(1) var a = 1, b = 1, c, d;
(2) c = ++a; alert(c);           // 2
(3) d = b++; alert(d);           // 1
(4) c = (2+ ++a); alert(c);      // 5
(5) d = (2+ b++); alert(d);      // 4
(6) alert(a);                    // 3
(7) alert(b);                    // 3
Почему код даёт именно такие результаты? 
(1) a = 1, b = 1, c = undefined, d = undefined
(2) c = 2, a = 2 (сначала прибавили 1 переменной а, затем присвоили с)
(3) d = 1, b = 2 (сначала присвоили значение b переменной d, затем увеличили b на 1)
(4) с = 2 + (2 + 1) = 5, a = 3 (увеличили a на 1, затем прибавили 2 и присвоили к с)
(5) d = 2 + 2 = 4, b = 3 (прибавили 2 к b и присвоили к d, затем b увеличили на 1)
(6) все, что делали с а, сохранилось
(7) все, что делали с b, сохранилось
